# Tunar Games - GitHub Pages

Suba o conteudo desta pasta na raiz do repositorio do GitHub Pages.

A estrutura correta precisa ficar assim:

- `index.html`
- `styles.css`
- `script.js`
- `assets/`
- `.nojekyll`

Nao suba uma pasta acima contendo `work/` ou `outputs/`, senao o GitHub Pages nao encontra o site completo.
